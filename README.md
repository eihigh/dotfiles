# dotfiles

## Steps
1. Install required commands: curl, make, ...
1. `git clone https://github.com/eihigh/dotfiles.git`
1. `sudo dotfiles/.utils/install-rcm`
1. `env RCRC=~/dotfiles/rcrc [HOSTRC=1] rcup`

## Optional
### install fzf
* `.fzf/install`

### install go tools
* `dotfiles/.utils/install-go-tools`
